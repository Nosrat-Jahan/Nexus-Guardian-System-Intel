# 🛡️ Nexus Guardian | System Intelligence v9.9.9

**Nexus Guardian** is a real-time hardware telemetry and system auditing suite. Engineered as part of a professional **BSc in CSE** portfolio, this module monitors CPU, RAM, and Disk I/O to detect anomalous patterns and potential resource-based security threats.

---

## ⚡ Core Functionality
- **Live Telemetry Stream**: Real-time hardware performance monitoring using `psutil`.
- **Heuristic Anomaly Detection**: AI-driven logic to identify sustained high-resource spikes indicative of malicious background processes.
- **Async Industrial UI**: A high-performance dashboard built with an industrial "Neon-Blue" aesthetic and zero-latency updates.
- **Modular API Architecture**: Implements a versioned (v1) REST API for telemetry data distribution.

## 🛠️ Tech Stack
- **Backend:** Python 3.14 / Flask (API Engine)
- **Telemetry:** Psutil (Hardware Abstraction Layer)
- **Frontend:** HTML5, CSS3 (Industrial Variable Architecture), JavaScript (ES6+ / Async-Fetch)
